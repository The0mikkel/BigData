Big Data

Big data er mange ting. Big data kan være de data som firmaer gemmer omkring dig mens du bruger deres service, eller det kan være de store datamængder vi har omkring vejret.

Big data er i dag et buzzword, som oftest forbindes sammen med misbrug af data som fører til at vi ikke har kontrol over vores personlige oplysninger.

En af de ting som gør big data så fremme i tiden, er at når big data analyseres, kan der skabes mange forbindelse, hvorfra man kan bestemme fx hvilke varer du måske ville kunne lide, og ramme mere rigtigt.

Et eksempel på hvor big data er blevet brugt, er hos Google. Google gemmer alle søgninger, og har udviklet modeller baseret på disse data, som kan vise hvornår en influenzaepidemi starter, og hvor langt spredningen er.
